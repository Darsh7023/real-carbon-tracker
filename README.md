# 🌱 EcoPulse - Carbon Footprint Awareness Platform

EcoPulse is an interactive web-based intelligence platform designed to help individuals track, understand, and optimize their daily carbon footprint emissions based on lifestyle parameters.

## 🚀 Technical Stack
- **Frontend Framework:** Semantic HTML5 & Responsive CSS3 architecture
- **Core Logic Engine:** Dynamic Client-side Component JavaScript
- **Deployment Platform:** High-fidelity Netlify cloud infrastructure
- **Validation Framework:** Automated Jest unit testing configurations

## 🎯 Features Implemented
- Real-time Carbon Emission Calculator covering Mobility, Home Energy, and Dietary inputs.
- Highly interactive UI with immediate visual feedback indicators.
- Personalized climate action suggestions based on carbon thresholds.
- Global Community Pledge Counter system for public validation.
